# Aerodynamic effects of fin length-to-diameter ratio on spherical projectile range and velocity decay

This repository contains **data + Python code** to reproduce core plots from my IB Physics Extended Essay (May 2025):

- Average **horizontal range vs fin length-to-diameter ratio**
- **Velocity decay vs fin length-to-diameter ratio**
- A simple **Spearman correlation** heatmap across key variables

> Preprint (Zenodo): **https://doi.org/10.5281/zenodo.18434071**
> Code (Zenodo): **https://doi.org/10.5281/zenodo.18443083**

## What’s inside

- `data/raw/` – tables transcribed from the EE (ranges, velocity decay, max velocity, uncertainties)
- `src/` – scripts to reproduce figures
- `outputs/` – figures generated by the scripts (created after you run them)

## Quickstart

### 1) Install Python packages
```bash
python -m venv .venv
source .venv/bin/activate   # macOS/Linux
# .venv\Scripts\activate  # Windows PowerShell

pip install -r requirements.txt
```

### 2) Generate all figures
```bash
python -m src.make_figures
```

Figures will appear in `outputs/figures/`.

## Notes on data

- Horizontal range trials are from EE Table 8.1.
- The two “outliers” (0.75× and 1.00×) have additional trials in EE Table 9.2.
- Velocity decay values are from EE Table 8.2.
- Max velocity values are from EE Table 15.1.
